"# face-mask" 
Project Description

The Face Mask Detector project uses Deep Learning to detect faces and determine whether the person is wearing a mask or not.

Main technology: TensorFlow/Keras, OpenCV.

Function:

Recognize faces from webcams.

Predict mask or mask-free labels.

Draw a frame around the face and display the label in the same color:

Green (mask)

Red (without mask)

The application is suitable for public environments, offices, schools to control mask wearing.
